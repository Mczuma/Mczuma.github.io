# Cybersecurity Labs

## Overview
A collection of cybersecurity exercises, experiments, and practical labs focused on learning defensive security, ethical hacking concepts, and system protection.

## Areas Covered
- Network security
- Linux security
- Vulnerability assessment
- Web security fundamentals
- Digital forensics concepts
- Security automation

## Tools Practiced
- Kali Linux
- Nmap
- Wireshark
- Burp Suite
- Metasploit Framework
- John the Ripper

## Purpose
These labs demonstrate continuous learning and hands-on practice in cybersecurity, with a focus on understanding attacks, improving defenses, and building secure systems.
