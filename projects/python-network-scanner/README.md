# Python Network Scanner

## Overview
A Python-based network scanning tool designed to discover devices and analyze network information. This project demonstrates practical networking and cybersecurity concepts.

## Technologies Used
- Python
- Networking
- Socket Programming
- Cybersecurity Concepts

## Features
- Scans network devices
- Identifies active hosts
- Performs basic network discovery
- Helps understand network visibility and security monitoring

## Purpose
This project was built to improve understanding of network reconnaissance, defensive security, and how security professionals analyze network environments.

## Future Improvements
- Add port scanning capabilities
- Integrate service detection
- Create a graphical user interface (GUI)
- Add reporting features
